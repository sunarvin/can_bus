# can_bus

#### 介绍

The LseeStudio shield uses MCP2515, MCP2551 and 16MHz oscillator.
It can reuse the SeeeStudio CAN Shield library.

NOTICE: The shiled in picture is an old edition and DOES NOT reserve SCL/SDA pins near AREF.
